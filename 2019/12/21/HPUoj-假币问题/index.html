河南理工大学oj题库的假币问题。
三分是这个题目的关键。

思路：
如果有三个硬币A，B，C：
随便两个放在天平上（加入是A，B），如果A=B，则C是假币；如果A<B，则说明小的A是假币。

再来看一下20个硬币的情况：

如图所示，每次将硬币三分，题目问的是最少多少次一定能分出结果，所以每次三分之后选择最大的一份就可以了。每次最大数量是上次结果n/3并向上取整。当n=0或者n=1时结束三分。

#include<iostream>
#include<cmath>
using namespace std;
int main(){
    long long n; // n比较大，所以定义long long int 
    while(1){
        cin>>n;
        if(!n) break;
        long long res = 0; // 记录次数 

        // 循环出结果 
        while(1)
        {
            if(n>=2)
            {
                n = ceil(double(n)/3.0); // n/3的结果向上取整 
                res++;
            }
            else break;
        }

        cout<<res<<endl;
    }
}
